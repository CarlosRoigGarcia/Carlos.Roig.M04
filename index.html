GIT

Introducció

El git es una eina de gestió de fitxers que ens permet fer modificacions manualment amb diverses ordres. Entre aquestes ordres tenima el: git init(ens permet crear un repestori git a un directori), git add name (ens permet afegir un arxiu al repestori git), git commit -m "Commit message"(Per desar els canvis) o git commit -a -m "Commit message"(Per fer un add i un commit a la vegada) i per ùltim el git status ens permet veure l'estat del repestori.

Branques

Qualsevol sistema de versions té un mecanisme per suportar el que es coneix com l'ús de branques. És un dels punts forts del Git. Cada vegada que fem un commit Git guarda una sèrie d'instantànies del treball preparat. Aquestes instantànies, guarden metadates com ara l'autor i el missatge amb el que acompanyem el commit.
- $ git branch testing
- $ git checkout testing
- $ git commit -a -m "made a ghange"
- $ git checkout master

Remot

Per treballar en remot, necessitem primer un repositori remot. Podem utitlizar moltes plataformes. En aquest cas treballarem amb GitLab. Amb la comanda $git remote add origin URL_GIT podem afegir un remot a un repositori. Origin és el nom del remot. Per lincar la branca master amb la branca master del origin, utilitzem $git branch --set-upstream master origin/master. Per descarregar un repositori, posem $git clone URL_GIT. Per pujar els commits, utilitzem git push. Per descarregar els commits, git pull.
